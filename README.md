# Running
Just run
```
./shrugapillar
```

You can also do this
```
sudo mv shrugapillar /usr/bin/
```
or
```
mv shrugapillar ~/.local/bin/
```

You can also use `cp` instead of `mv`

# Compiling
It's very shrimple
```
gcc main.c -o shrugapillar
```
